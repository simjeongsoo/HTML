## HTML 목록 만들기
* 연관 있는 항목(item)들을 나열 한 것을 의미
* HTML 목록은 ‘순서없는 목록(unordered list)’ 과 ‘순서 있는 목록(ordered list)’ 으로 구분됨
* 블록 레벨 요소를 만드는 태그 

### 순서없는 목록(unordered list) : 
```
<ul>
	<li>con</li>
	<li>con</li>
	<li>con</li>
	<li>con</li>
</ul>
```

### 순서 있는 목록(ordered list) : 
```
<ol>
	<li>con</li>
	<li>con</li>
	<li>con</li>
	<li>con</li>
</ol>
```
